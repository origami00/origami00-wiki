## 项目概述

**origami00-wiki** 是一个基于 React 18 + Vite 的个人主页/维基网站，采用「Soft Minimalism」磨砂玻璃设计风格，中文界面，包含瀑布流卡片布局首页和照片墙页面。

**技术栈**：React 18.3.1 / TypeScript 6.0.3 / React Router 6.30.3 / Vite 5.4.21 / Lucide React 1.8.0
